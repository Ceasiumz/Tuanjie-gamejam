# 项目说明 V1.0
## 项目简单介绍
- 本项目为2024年12月Unity Game Jam 作品 https://unity.cn/gamejam2024
- 主题 "It never ends" 
- 开发周期七天 应用团结引擎开发
- 开发内容为核心玩法为21点的桌面卡牌游戏
